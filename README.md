## 🔧 Features

✔ Check Pi Node status  
✔ Monitor node connectivity  
✔ Fast and lightweight tool  
✔ Simple UI for quick usage  
✔ Fully browser-based (no install required)

## ⚡ What It Does

- 🔗 Verifies node availability  
- 🌐 Checks connection response  
- 📊 Helps diagnose node issues  
- ⚙️ Useful for Pi Node operators

### 📥 Clone Repository

```bash
git clone https://github.com/sanjeev28/Pi-Node-Tool.git
cd Pi-Node-Tool
```

### 🌐 Run Tool

Open `index.html` in your browser.
